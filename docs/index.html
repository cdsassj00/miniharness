단계별로 드러냅니다

Claude Code·Codex 는 결과만 보여주지만, CDSA Harness 는 컨텍스트·토큰·판단·도구·되먹임까지 에이전트의 모든 단계를 펼쳐 보여줍니다.

npm 으로 설치 실행파일 다운로드

made by CDSA · MIT · 의존성 0개

Install

30초 설치 — 둘 중 하나

① Node.js 가 있다면

개발자·일반 사용자

npx cdsa-harness
# 또는 전역 설치
npm install -g cdsa-harness
② Node 없이 · 폐쇄망

파일 하나만 받아 실행

cdsa-harness-win.exe   · Windows
cdsa-harness-macos     · macOS
cdsa-harness-linux     · Linux
Why

속을 다 보여주는 하네스

시키면 AI 가 스스로 파일을 보고·읽고·고치며 일합니다. 그 과정을 한 단계도 숨기지 않아요.

입력 → 컨텍스트 구성 → LLM 호출 → 도구 판단 → 실행/승인 → 결과 되먹임 → 반복

모델에 보내는 메시지, 추정 토큰, 실제 응답시간·토큰 사용량, 도구 호출, 결과를 다시 모델에 넣는 ‘되먹임’까지 그대로 표시합니다.
