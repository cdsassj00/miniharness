⚡ 30초 설치

① Node.js 가 있다면 
npx cdsa-harness
# 또는
npm install -g cdsa-harness
② Node 없이 (폐쇄망)

Releases 에서 OS 파일 하나만 받아 실행.

cdsa-harness-win.exe   (Windows)
cdsa-harness-macos     (macOS)
cdsa-harness-linux     (Linux)

🧠 왜 특별한가

Claude Code·Codex 는 결과만 보여주지만, CDSA Harness 는 속을 다 보여줍니다 — 어떤 컨텍스트를 보내는지, 토큰·응답시간, 모델의 판단, 도구 실행, 그리고 결과를 다시 모델에 넣는 되먹임까지. 
입력 → 컨텍스트 구성 → LLM 호출 → 도구 판단 → 실행/승인 → 결과 되먹임 → (반복)
